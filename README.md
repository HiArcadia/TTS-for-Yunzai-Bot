#### 此为Yunzai-Bot云崽QQ机器人的插件，需要依赖云崽机器人来实现功能

### 插件功能
用于实现文字转语音。
![音色列表](./soundlist.jpg)
### 使用方法
根据你的云崽版本下载对应版本插件，V2放进/lib/example文件夹，V3放进plugins/example文件夹
